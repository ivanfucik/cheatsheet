CSS vlastnosti

Vlastnost Popis Příklad Odkaz
background-color Nastaví barvu pozadí elementu thead {
  background-color: #0087e0;
} 		MDN
border Ohraničí element čárou s různými styly a barvou table, th, td {
  border: 1px solid black;
} 		MDN
border-collapse Sloučí ohraničení buňěk a tabulky do jedné čáry table {
  border-collapse: collapse;
} 		MDN
border-radius Zaoblí ohraničení prvku td {
  border-radius: 20px;
} 		MDN
font-family Nastaví font pro daný element. Je možné využít více možností, pokud by jedna nebyla podporována prohlížečem. body {
  font-family: Arial, Helvetica, sans-serif;
} 		MDN
