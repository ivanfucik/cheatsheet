CSS vlastnosti

Vlastnost Popis Příklad Odkaz
background-color Nastaví barvu pozadí elementu thead {
  background-color: #0087e0;
} 		MDN
border Ohraničí element čárou s různými styly a barvou table, th, td {
  border: 1px solid black;
} 		MDN
border-collapse Sloučí ohraničení buňěk a tabulky do jedné čáry table {
  border-collapse: collapse;
} 		MDN
border-radius Zaoblí ohraničení prvku td {
  border-radius: 20px;
} 		MDN
display Umožňuje změnit zobrazení prvků (například v řadě). MDN
font-family Nastaví font pro daný element. Je možné využít více možností, pokud by jedna nebyla podporována prohlížečem. body {
  font-family: Arial, Helvetica, sans-serif;
} 		MDN
float Vyjme prvek z toku a umístí ho např. do leva. .class {
  float: left;
} 		MDN
font-size Nastaví velikost fontu. Lze využívat různé velikosti např. mezi stavy u tlačítek. p {
  font-size: 2em;
} 		MDN
text-decoration Určuje například zrušení podržení u odkazů. a {
  text-decoration: none;
} 		MDN
transition Definuje přechod mezi různými stavy (například :hover a :active) .activity:hover {
  background-color: #fff;
  color: #5495ff;
  transition: color .1s ease-in, background-color .1s ease-in;
} 		MDN

CSS pseudo prvky

Pseudo-prvek Popis Příklad Odkaz
::before Používá se s vlasností content k doplnění kosmetických prvků k textu. .breadcrumb li+li::before {
  padding: 10px;

} 		MDN

Návody

Dotaz Popis Příkladd Odkaz
Jak vycentrovat tabulku? Pravý a levý margin se nastaví na auto. Tím se vycentruje objekt do prostřed stránky nebo do prostřed nadřazeného prvku. .center {
  margin-left: auto;
  margin-right: auto;
} 		w3schools
Následující selektor jdoucí po sourozenci (Next-sibling combinator +) Pomocí znaku + se označí selektor jdoucí po jiném selektoru, na který se aplikuje daný styl. .breadcrumb li+li::before {
  padding: 10px;
  content: ">";
}

Na prvek <li> jdoucí za jiným <li> se aplikuje styl.
